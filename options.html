Admin Toggler - Settings

The root domain this extension operates on (no protocol, no trailing slash).
The path segment prepended to URLs when toggling (no slashes).
Each line: a regex matched against the URL path, then =>, then a replacement string.
Capture groups ($1, $2, …) work in the replacement. Rules are checked top-to-bottom; first match wins.
If no rules match, the default behaviour (prepend /prefix) is used.
